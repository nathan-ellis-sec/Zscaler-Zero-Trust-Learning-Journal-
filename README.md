# Zscaler-Zero-Trust-Learning-Journal

This repository documents my journey into Zero Trust Security Engineering using Zscaler technologies (ZIA, ZPA, ZDX). These notes come from my studies for the ZDTA and ZDTE certifications.

## Why This Repo Exists
I'm transitioning from SOC Analyst to Zero Trust Engineer. This repo shows my learning process. 

## What's Inside
- **ZDTA Notes** — Concepts, architecture, and policy logic.
- **ZDTE Notes** — Deployment, troubleshooting, and best practices.
- **Architecture Diagrams** — Text-based diagrams showing ZIA/ZPA flows.

## Topics Covered
- Zero Trust fundamentals
- ZIA and ZPA architecture
- Policy creation and evaluation
- Identity-driven access
- Device posture and continuous verification
